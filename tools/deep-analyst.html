Agent Execution Flow
The Deep Analyst ingests a target account and autonomously executes all 6 phases, producing deal-ready intelligence as structured JSON output.
1

Target Account Ingestion

Agent receives account name, industry vertical, and known signals. Validates against existing CRM data if available.

2

Deep Research Execution

Perplexity-powered research agent generates comprehensive account intelligence: financials, tech stack, org structure, competitive landscape, and strategic priorities.

3

Persona and Committee Mapping

Identifies 15-25 key stakeholders, maps buying committee structure, assigns P5-P1 priority scores, and generates persona-specific messaging angles.

4

Intent Signal Enrichment

Cross-references 6sense buying stage data, ZoomInfo intent signals, and technographic triggers. Produces confidence-scored timing windows.

5

JSON Intelligence Output

Structures all findings into machine-readable JSON: KPIs, alerts, persona maps, competitive positioning, and recommended plays.

6

Downstream System Handoff

JSON output feeds Salesloft cadences, CRM fields, and rep dashboards. Human validates, then activates play with pre-built messaging.

Technical Architecture
Component breakdown and integration points for the autonomous research pipeline.

Input Layer

  • Account name + industry vertical
  • Known intent signals (optional)
  • CRM opportunity data (optional)
  • Competitive context parameters

Research Engine

  • Perplexity API for deep research
  • Structured prompt chains (6 phases)
  • Confidence scoring at L1/L2/L3 levels
  • Source validation and citation tracking

Output Schema

  • JSON: KPIs, alerts, personas, plays
  • Battle card generation (PDF export)
  • CRM field mapping (Salesforce)
  • Cadence trigger definitions

Integration Points

  • 6sense: buying stage + intent scores
  • ZoomInfo: contact enrichment
  • Salesloft: cadence activation
  • Salesforce: opportunity + field updates
